Idea Propuesta

Aplicación de opciones múltiples

  • Herramientas de desarrollo

    El uso de las nuevas tecnologías, con el fin de llevar a más sectores

  • Duración del proyecto (Tiempo)

    Se espera que la aplicación perdure y se potencie con el paso del tiempo

  • Anális de perfiles de usuarios

    Continuar diversificando opciones para mayor abarcamiento del problema

Soluciones para la idea
  • Se propone un aplicativo en que se remunere o se den cupones a las personas por hacer uso de medios de transporte más ecológicos y sostenibles como lo son: el metro, la bicicleta, patinetas
  • Se propone un sitio web en alianza con el metro, el cual conceda puntos que son redimibles cada vez que se haga uso de el sistema de transporte del metro
  • Se propone un aplicativo para todo el sistema de buses en la ciudad que sean ecológicos, el cual ofrezca bonos y cada cantidad de viajes otorgue viajes gratis y cupones de descuento
Solución Agregada de valor
  • Idea 1

    Las remuneraciones y alianzas que se pueden establecer por hacer uso de estas alternativas de transporte sostenibles.

  • Idea 2

    La alianza con el metro para fomentar el uso del transporte sostenible y ecológico con el medio ambiente. Por otro lado, los incentivos van a dar la oportunidad de llamar más la atención de la gente para usar este tipo de transportes.

  • Idea 3

    Mayor cobertura para el transporte de la ciudad, ofreciendo así llegar a más personas e impulsando a las empresas de transporte a modernizarse para ofrecer el servicio.
